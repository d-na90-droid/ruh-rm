# قسم العلاقات العامة والإعلام — بوابة المعلومات والأرشيف المؤسسي

منصة ويب متكاملة لإدارة وأرشفة أنشطة قسم العلاقات العامة والإعلام، مبنية بـ HTML/CSS/JavaScript خالص بدون أي خادم خلفي.

🔗 **الموقع المنشور:** [https://d-na90-droid.github.io/ruh1/](https://d-na90-droid.github.io/ruh1/)

---

## المميزات

- 🔐 نظام تسجيل دخول بمستويات صلاحيات متعددة
- 📊 إحصائيات تلقائية تُحسب من بيانات الأقسام
- 🔍 بحث ذكي فوري مع تمييز النتائج
- 📄 رفع واستعراض وتحميل ملفات PDF مباشرة
- 🔽 فلترة تصاعدية/تنازلية وتصنيف حسب السنة
- 💾 حفظ تلقائي في المتصفح (localStorage + IndexedDB)
- 📱 تصميم متجاوب مع جميع الأجهزة

## التقنيات المستخدمة

- **HTML5 / CSS3 / JavaScript** — بدون أي framework
- **localStorage** — لحفظ البيانات النصية
- **IndexedDB** — لحفظ ملفات PDF بدون حد للحجم
- **Font Awesome 6** — للأيقونات
- **Google Fonts (Tajawal)** — للخط العربي
- **GitHub Pages** — للنشر المجاني

## الرفع والتحديث

```bash
# أي تعديل على index.html في GitHub يُحدّث الموقع تلقائياً خلال دقيقة
```

1. افتح `index.html` في الـ Repository
2. اضغط ✏️ Edit
3. عدّل ثم اضغط **Commit changes**

---

## ملاحظات مهمة

- البيانات تُحفظ في **متصفح كل مستخدم** بشكل منفصل — التعديلات لا تنتقل بين الأجهزة
- لنقل البيانات: استخدم زر **تصدير البيانات** ثم **استيراد بيانات** في الجهاز الآخر
- ملفات PDF تُخزن في IndexedDB ولا تُصدَّر مع ملف JSON
